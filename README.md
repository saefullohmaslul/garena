## Instalation

Make sure you have already install nodejs and npm!

```bash
# install dependencies
npm install

# run project in dev
npm run start:dev
```

And then goto [localhost:3000](localhost:3000)

## Build

To build this application using webpack

```bash
npm run build
```

And goto dist folder, it will bundle all you need.
